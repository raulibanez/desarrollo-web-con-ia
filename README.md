# Desarrollo de aplicaciones web · SMR

Curso de desarrollo web (HTML, CSS y JavaScript) usando la IA como herramienta de aprendizaje y construcción.

**🌐 Sitio publicado:** <https://raulibanez.github.io/desarrollo-web-con-ia/>

---

## Sobre el curso

Asignatura del ciclo **SMR (Sistemas Microinformáticos y Redes)** centrada en aprender desarrollo web desde cero — sin memorizar etiquetas, entendiendo qué se puede construir y cómo pedírselo a la IA.

- **8 temas** · 90 minutos por sesión
- **Formato:** teoría + práctica en vivo
- **Herramientas:** VS Code · Cursor · ChatGPT · Claude

## Temas

1. **HTML** — la estructura de una página
2. **Componentes UI** — navbars, drawers, toasts, pricing tables…
3. **CSS** — colores, tipografía, espacios
4. **CSS avanzado** — flexbox, grid, responsive
5. **JavaScript** — variables, funciones, lógica
6. **DOM y eventos** — JS en el navegador
7. **APIs y datos** — fetch, asincronía
8. **Despliegue** — de tu portátil a una URL pública

> Consulta el [índice del curso](https://raulibanez.github.io/desarrollo-web-con-ia/) para ver qué temas están publicados.

## Estructura del repositorio

- [`index.html`](index.html) — portada con el índice de temas
- [`Tema 1 - HTML.html`](Tema%201%20-%20HTML.html) — slides del tema 1
- [`Tema 2 - Componentes UI.html`](Tema%202%20-%20Componentes%20UI.html) — slides del tema 2
- [`Tema 3 - CSS.html`](Tema%203%20-%20CSS.html) — slides del tema 3
- [`deck-stage.js`](deck-stage.js) — web component reutilizable que gestiona las presentaciones (navegación con teclado, miniaturas, escalado, impresión a PDF…)

## Cómo ver los temas

Cada tema es una presentación HTML autónoma construida con el web component `<deck-stage>`.

**Atajos de teclado:**

| Tecla | Acción |
|---|---|
| `←` `→` · `PgUp` `PgDn` · `Space` | Navegar entre slides |
| `Home` `End` | Ir al inicio / final |
| `1`–`9` | Saltar a un slide concreto |
| `R` | Reset al primer slide |
| `T` | Mostrar / ocultar miniaturas |

En móvil/tablet, tocar la mitad izquierda o derecha de la pantalla navega entre slides.
